# 2018 WhoIs Level Test CTF

## 소개

2018 WhoIs Level Test CTF는 2018년 신입 회원을 대상으로 진행한 레벨 테스트 CTF입니다.

### 운영

강준구, 권혁주, 김형찬, 이시훈, 이재현

#### 운영 지원

2018 WhoIs Level Test CTF는 Naver D2의 지원으로 운영되었습니다.
